# StatusBadge

## Introduction

StatusBadges can act as labels in applications by utilizing the isAdmin prop. When isAdmin is true, the StatusBadge is rendered a checkmark label indicating privileged or admin status. When isAdmin is false (default), the StatusBadge have the dismiss mark label appearance.

## Basic Usage

Firstly, you have to import `StatusBadge` atom so that you can start to use it

```
	import { StatusBadge } from "@atoms/StatusBadge";

	or

	import { StatusBadge } from "@atoms/index";

	<StatusBadge/>
	<StatusBadge isAdmin={true} />

```

<StatusBadge />
<StatusBadge isAdmin={true} />

## Sizes

StatusBadges can be customized in size using the 'size' prop, which accepts five different options: `'xs', 'sm', 'md', 'lg', and 'xl'`, and the default size is `'md'`. This allows you to easily adjust the size of the StatusBadge to suit your needs.

```
	<StatusBadge  size='xs' />
    <StatusBadge  size='md' />
    <StatusBadge  size='lg' />
    <StatusBadge  size='xl' />
    <StatusBadge  size='sm' />

	<StatusBadge  size='xs' isAdmin={true}/>
    <StatusBadge  size='sm' isAdmin={true}/>
    <StatusBadge  size='md' isAdmin={true}/>
    <StatusBadge  size='lg' isAdmin={true}/>
    <StatusBadge  size='xl' isAdmin={true}/>
```

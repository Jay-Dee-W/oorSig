# Radial Progress Bar

## Introduction

The `Radial Progress Bar` component displays a circular progress bar.

## Props

### `progress`: `number` (required)

- The progress value of the progress bar (0-100).

### `radius`: `number` (optional)

- The radius of the progress bar circle. The default value is 100.

### `color`: `string` (optional)

- The color of the progress bar. The default color is primary-200.

## Basic Usage

Here's an example of how to use the `Radial Progress Bar` component:

```jsx
import React from 'react';
import { RadialProgressBar } from '@atoms/RadialProgressBar';

<RadialProgressBar progress={60} radius={80} color="green" />;
```

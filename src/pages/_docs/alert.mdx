# Alert

## Introduction

The `Alert` component displays a dismissible alert message with different color variants.

## Props

### `message`: `string`

- The alert message to be displayed.

### `variant`: `'success' | 'danger' | 'info'` (optional)

- The color variant of the alert. Defaults to `'info'`.

### `onDismiss`: `() => void` (optional)

- A callback function that will be called when the dismiss button is clicked.

## Basic Usage

Here's an example of how to use the `Alert` component:

```jsx
import React from 'react';
import { Alert } from '@atoms/Alert';

<Alert message="This is an information alert" variant="info" />;
```

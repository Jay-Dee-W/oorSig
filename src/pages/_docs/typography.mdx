# FontSize and lineHeight

In the OORSIG design system, font sizes are defined with the following values: xs,
sm, base, lg, xl, 2xl, 3xl, 4xl, 5xl, 6xl. These values can be used as the values
for the fontSize tag when applying typography styles to your text elements.

# Using lineHeight

In the OORSIG design system, we have defined a set of line heights that correspond to our font sizes. These line heights, which are named xs, sm, base, lg, xl, 2xl, 3xl, 4xl, 5xl, and 6xl, are designed to make it easier for you to remember which line height to use with a particular font size.

To achieve the best visual results and improve the readability of your text, it is important to use the appropriate line height for each font size. This means that when you use a particular font size, you should also use the corresponding line height.

For example, if you are using a font size of base, you should also use a line height of base to ensure that your text is properly spaced and easy to read. Similarly, if you are using a font size of 3xl, you should use a line height of 3xl.

By using the appropriate line height for each font size, you can ensure that your text is legible and visually appealing, while maintaining a consistent look and feel across your design.

# Using Typography component

The Typography component is used to render text content in a specific font size and HTML element. The component accepts a size prop, which determines the font size and line height based on the defined values in the OORSIG design system. The variant prop is used to specify the HTML element in which the content should be rendered, such as h1, h2, p, span, and so on.

In addition to the size and variant props, the Typography component also accepts any props available in the xstyled library, such as p, m, color, and backgroundColor. These props can be used to further customize the appearance of the text.

## Examples

<Typography variant="span" color='gray-100'>
  Lorem ipsum dolor...
</Typography>
<Typography variant="span" color='gray-200'>
  Lorem ipsum dolor...
</Typography>
<Typography variant="span" color='red'>
  Lorem ipsum dolor...
</Typography>

## Codes

```
<Typography variant="span" color='gray-100'>
  Lorem ipsum dolor...
</Typography>
<Typography variant="span" color='gray-200'>
  Lorem ipsum dolor...
</Typography>
<Typography variant="span" color='red'>
  Lorem ipsum dolor...
</Typography>
```
It's important to note that the size prop must be used in conjunction with an appropriate variant prop. For example, if the size prop is set to 6xl, the variant prop should be set to h1, which is the appropriate HTML element for the largest heading size. The defined font sizes in the OORSIG design system are: xs, sm, base, lg, xl, 2xl, 3xl, 4xl, 5xl, and 6xl, and they are named to correspond with the HTML element names for easier reference.

Overall, the Typography component is a flexible and customizable way to display text content in a consistent and visually appealing manner within a project that uses the OORSIG design system.

# Searchable Dropdown

## Introduction

The `Searchable Dropdown` component is a custom dropdown implementation using `downshift` and `@xstyled/emotion` libraries.

## Props

### options (required)

An array of objects representing the options in the dropdown.
Each object should have a label, value and imgSrc property. For example:

```
const options = [
  { label: 'Option 1', value: 'option1', imgSrc: 'src1' },
  { label: 'Option 2', value: 'option2', imgSrc: 'src2' },
  { label: 'Option 3', value: 'option3', imgSrc: 'src3' },
];
```

### placeholder (optional)

A string that specifies the placeholder text to be displayed in the dropdown when no option is selected. Default value is "Search option".

### label (optional)

A string representing the label of the dropdown. This label will be displayed above the dropdown. Default value is "Searchable Dropdown".

## Basic Usage

This is just a basic usage and customization example.

```jsx
import React from 'react';
import { SearchableDropdown } from '@atoms/index';

const options = [
  { label: 'Option 1', value: 'option1' },
  { label: 'Option 2', value: 'option2' },
  { label: 'Option 3', value: 'option3' },
];

<SearchableDropdown
  options={options}
  placeholder="Search Team"
  label="Select Team"
/>;
```

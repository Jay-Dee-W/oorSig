# Avatar

## Introduction

Card are graphical elements that show visual form representing a card with header

## Basic Usage

Firstly, you have to import `Card` atom so that you can start to use it

```
	import { Card } from "@atoms/Card";
	or
	import { Card } from "@atoms/index";
	<Card />
```

## Sizes

Avatar can be customized in size using the 'size' prop, which accepts four different options: `10, 20` and the default size is `10`. This allows you to easily adjust the size of the avatar to suit your needs.

```
    <Card bg="white" p={4} borderRadius={4} boxShadow="lg">
      <h2>Card title</h2>
      <p>Some card content goes here</p>
    </Card>
```

# Weekly Progress

## Introduction

The `WeeklyProgress` omponent is a progress card that displays total PRs and average weekly goals.

## Props

### `progress`: `number`

- The progress value of the progress bar (0-100).

### `total`: `number`

- The total number of PRs.

### `avg`: `number`

- The average weekly goal.

## Basic Usage

Here's an example of how to use the `WeeklyProgress` component:

```jsx
import React from 'react';
import { WeeklyProgress } from '@atoms/WeeklyProgress';

<WeeklyProgress total={5} avg={3} progress={10} />;
```

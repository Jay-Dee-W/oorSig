# Avatar

## Introduction

Avatar are graphical elements that show visual form representing a user oneself and one's identity wished

## Basic Usage

Firstly, you have to import `Avatar` atom so that you can start to use it

```
	import { Avatar } from "@atoms/Avatar";
	or
	import { Avatar } from "@atoms/index";
	<Avatar/>
```

## Sizes

Avatar can be customized in size using the 'size' prop, which accepts four different options: `10, 20` and the default size is `10`. This allows you to easily adjust the size of the avatar to suit your needs.

```
	<Avatar size='10' />
	<Avatar size='20' />
	<Avatar size='30' />
```

<x.div display="inline-flex" gap={6}>
  <Avatar size="10" />
  <Avatar size="20" />
  <Avatar size="30" />
</x.div>

## Src

Avatar can be customized in size using the 'src' prop, which accepts image url

```
	<Avatar borderColor="#000" size='20' src='https://gravatar.com/avatar/4213bb0627246966c1c4341da2d8544e?s=200&d=robohash&r=x' />
```

<x.div display="inline-flex" gap={6}>
  <Avatar
    size="20"
    src="https://gravatar.com/avatar/4213bb0627246966c1c4341da2d8544e?s=200&d=robohash&r=x"
  />
</x.div>

## systemProps

In addition to the size prop, you can further customize avatars by passing system props such as borderColor , border and other as props. These additional props provide more flexibility in styling, with the default borderColor being set to white.

<Avatar
  size="20"
  alt="Avatar"
  src="https://gravatar.com/avatar/4213bb0627246966c1c4341da2d8544e?s=200&d=robohash&r=x"
/>

</MDXWrapper>
